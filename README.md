# MyRayCaster

Basic raycaster made using the [CSFML](https://github.com/SFML/CSFML) library.

## Compilation

```shell
make
```

## Usage

```shell
./raycaster
```

## Controls

`Arrow Up/Down` => Move forward/backward \
`Arrow Left/Right` => Turn left/right \
`Comma (,)/Period (.)` => Increase/Decrease graphics
